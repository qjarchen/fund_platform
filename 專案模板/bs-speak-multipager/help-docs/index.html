Features List of Template

Speak multipager template is featured rich template. It can be used for many kinds of web works.Some of its features are described below:

  • Bootstrap 3.1.1
  • Two Color Variants (Blue & Brown)
  • Two Home Page Variants ( In Blue & Brown )
  • Two About Us Variants ( In Blue & Brown )
